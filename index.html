关于我们

这是一个使用 Bootstrap 5 构建的演示 SPA。通过监听 hashchange 事件实现无刷新的页面切换，所有页面内容写在一个 HTML 文件中。

你可以在此基础上添加更多页面、集成 API 调用、或者引入 Vue/React 等框架来增强功能。
